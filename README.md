# Generador de Rutinas de Estiramiento — Prototipo Wireframe

Prototipo de interfaz móvil para una aplicación de rutinas de estiramiento, generado con asistencia de inteligencia artificial. El objetivo del prototipo es representar la estructura, flujo y componentes principales de la pantalla sin colores, imágenes ni estilos decorativos.

---

## Descripción del proyecto

La aplicación muestra un ejercicio de estiramiento aleatorio con instrucciones breves y un temporizador integrado de 30 segundos. El usuario puede iniciar el temporizador, generar un nuevo ejercicio aleatorio o saltar al siguiente.

### Funcionalidades prototipadas

- Tarjeta de ejercicio con nombre, categoría e instrucciones paso a paso
- Temporizador visual de 30 segundos con barra de progreso
- Botón para iniciar el temporizador
- Botones para generar ejercicio aleatorio y saltar
- Barra de navegación inferior con 4 secciones: Inicio, Ejercicios, Progreso y Ajustes

---

## Herramientas utilizadas

| Herramienta | Versión / Detalle | Uso |
|---|---|---|
| **Claude (Anthropic)** | Claude Sonnet 4.6 | Ideas e instrucciones de como realizar un prototipado
| **HTML5** | Estándar | Estructura y marcado del wireframe |
| **CSS3** | Variables nativas | Estilos, layout y adaptación a modo claro/oscuro |
| **Tabler Icons** | v3.x (CDN) | Iconografía outline (play, refresh, home, etc.) |

## Notas

- Este es un prototipo de **baja fidelidad**. No incluye lógica funcional ni datos reales.
- El objetivo es validar la estructura de pantalla y el flujo de usuario antes de pasar al diseño de alta fidelidad.
- El código fue generado con IA y revisado manualmente antes de ser subido al repositorio.
